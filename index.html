爆焼けスコアの計算式

スコア構成（合計100点）

指標配点理想値
全体雲量40点20〜50%（適度な雲）
湿度25点40%以下（低いほど透明度高）
視程15点20km以上（大気が澄んでいる）
高層雲（巻雲）10点多いほど良い（光を広域に拡散）
雨上がりボーナス10点直近3時間に0.5〜15mm

全体雲量スコア

雲量 0%　　　→ 0点（キャンバスなし） 雲量 20〜50% → 満点（光を反射する適度な雲） 雲量 50〜80% → 逓減（厚すぎると光が通らない） 雲量 80%以上 → 0点（太陽光が届かない）

湿度スコア

湿度 40%以下 → 満点 湿度 40〜80% → 線形に逓減 湿度 80%以上 → 0点

判定基準

スコア判定
70〜100🔥 爆焼けチャンス！
50〜69🌅 期待できる
30〜49🌤 少し期待
0〜29😐 今日は厳しいかも

夕焼けのしくみ：太陽が地平線に沈む直前、大気を斜めに通過した赤い光が雲に反射することで起きる。雲が少なすぎると反射面がなく、多すぎると光が遮られる。「適度な雲が西空にあり、頭上は晴れている」状態が最適。高度8〜12kmの薄い巻雲は光を遮らずに広く拡散させるため、爆焼けに大きく効く。

データソース

日没時刻・雲量・湿度・視程・降水量Open-Meteo API（日没前後1時間の値）

🌈 虹色コントレイルスコア

空港を選択してください

虹色コントレイルの計算式

スコア = 高層湿度スコア × 太陽仰角係数

高層湿度スコア（300hPa ≈ 9,500m） RH < 40% → 0 RH 40〜60% → 0〜30（線形） RH 60〜80% → 30〜70（線形） RH 80〜100% → 70〜100（線形） 太陽仰角係数 仰角 < 10° → 0（太陽が低すぎ） 仰角 10〜15° → 0〜0.5 仰角 15〜35° → 1.0（ゴールデンゾーン） 仰角 35〜50° → 1.0〜0.4 仰角 > 50° → 0.4〜0（昼間は厳しい）

なぜこの条件か

虹色飛行機雲（コントレイル彩雲）は回折現象。生まれたての飛行機雲の氷晶が均一サイズのとき、太陽から10〜40°の角度で撮影すると虹色に見える。上空300hPa（約9,500m / FL310）の湿度が高いほど氷晶が豊富に形成され、彩雲が出やすい。

撮影のコツ

太陽からこぶし2〜4個（20〜40°）離れた方向を向いて空を見る。太陽を直視せず、その「すぐ隣」に飛行機雲が来た瞬間がシャッターチャンス。

データソース

300hPa 相対湿度Open-Meteo（1時間ごと予報）
太陽仰角SunCalc.js（時刻・座標から計算）

🌈 虹色ベイパー

RWY32 離陸→左旋回ベイパーの計算式

総合スコア = 湿度スコア×0.7 ＋ 時間帯スコア×0.3

湿度スコア（スプレッド = 気温 − 露点） ≤ 1.0°C → 100点 ≤ 3.0°C → 線形に逓減 > 3.0°C → 0点 時間帯スコア（JST） 7〜 9時 → 100点（朝日が東・逆光ベイパー最適） 9〜11時 → 80点（朝の光 良好） 6〜 7時 → 40点（夜間門限前） 11〜14時 → 30点（太陽が高くなる） その他 → 10点

このシナリオ

RWY32（北北西方向）離陸後に左旋回すると機体は南南西方向へ転じる。猪名寺・塚口・伊丹市南部はこの旋回後ルート直下にあたり、低高度＋フラップ展開中の機体を正面から狙える。旋回中の荷重増加でベイパーが強調される。

推奨撮影スポット

スポット見え方
猪名寺（猪名川堤防）旋回後の機体を正面〜北側から
塚口（JR塚口駅周辺）旋回後の機体を西側から
伊丹市南部（猪名川東岸）上昇中の機体を東側から

データソース

気温・露点aviationweather.gov（METAR、伊丹 RJOO）
時刻端末の現在時刻（JST変換）

🌫️ ベイパースコア

空港を選択してください

ベイパー発生の計算式

スプレッド（温度差）

スプレッド = 気温 − 露点温度

判定基準

スプレッド判定
0〜1°C✅ ほぼ確実に出る
1〜3°C⚡ 出る可能性あり
3°C 超❌ まず出ない

なぜスプレッドが小さいと出るのか

着陸時に翼端渦・フラップ後縁で局所的な気圧降下が発生する。この圧力低下により温度がさらに下がり、露点に達して空気中の水分が凝結し、ベイパーとして可視化される。

朝方・雨上がり直後・海沿い → 湿度が高くなりやすい
大型機・フル高揚力装置展開の着陸時 → 渦が強く出やすい

データソース

気温・露点温度aviationweather.gov（METAR、30分更新）

🌦️ レインボースコア

空港を選択してください

レインボースコアの計算式

スコア = 降水スコア × 太陽仰角係数 × 雲量係数

降水スコア（直近2時間の合計降水量） 0mm → 0（雨なし = 虹なし） 0〜1mm → 40〜70（線形） 1〜5mm → 70〜100（線形） 5mm超 → 100（十分濡れた空気） 太陽仰角係数（低いほど大きな虹） 仰角 > 42° → 0（虹の円弧が地面の下に沈む） 仰角 35〜42° → 0.5 仰角 15〜35° → 0.8 仰角 5〜15° → 1.0（ベスト：大きな半円虹） 仰角 0〜 5° → 0.3（太陽が低すぎて霞む） 雲量係数（太陽が出ているか） 雲量 < 50% → 1.0 雲量 50〜70% → 0.5 雲量 > 70% → 0.2

なぜこの条件か

虹は太陽光が空気中の水滴で屈折・反射して起きる。水滴は太陽の反対方向・仰角42°の円弧に見える。太陽が高すぎると42°の円弧が地面の下に沈んで見えなくなる。朝夕の雨上がりが最高条件。

データソース

降水量Open-Meteo（1時間ごと、当日分）
雲量Open-Meteo（hourly cloud_cover）
太陽仰角SunCalc.js（時刻・座標から計算）

✈️ ワンフォースコア

ワンフォースコアの計算式

追い風成分

追い風成分 = 風速 × cos(風向 − 140°) 風向 = METAR の風向（度） 風速 = METAR の風速（kt） 140° = RWY32（逆方向 RWY14）の滑走路方位

ワンフォースコア

スコア = 追い風成分 × 8　（0〜100でクランプ）

判定基準

追い風成分判定
12.0 以上🔴 断続継続で RWY14 運用の可能性大
9.0〜11.9🟡 ランチェン可能性あり
9.0 未満🟢 RWY32 通常運用

データソース

風向・風速aviationweather.gov（METAR、伊丹 RJOO）
滑走路方位RWY14/32 磁方位 140°/320°
